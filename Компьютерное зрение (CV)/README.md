# Распознование шахматной доски по фотографиям

**Задача:** В качестве тестового задания предлагается проблема поиска
шахматной доски. Вам дается два набора данных (xtrain, ytrain) и xtest.
Формат данных – массив NumPy. Требуется построить модель одного на
Ваш выбор алгоритма машинного обучения, обучить на xtrain и
предсказать ytest для xtest

**Решение:** Перед нами задача регрессии: нам необходимо пресдказать массив из 8 чисел. Задача была решена построением свёрточной нейросети, используя 500 фотографий из 15000 тренировочной выборки, с 8 нейронами на выходе. В качестве валидационной выборки взяли 100 фотографий. Тестовая выборка состоит из 5 фотографий. Предсказания для тестовой выборки в конце тетради.
*MAE тренировочной выборки:* 0.0065
*MAE валидационной выборки:* 0.012
